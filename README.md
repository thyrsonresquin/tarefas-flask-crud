# Gerenciador Tarefas com Flask

## Descrição
Este é um projeto de um gerenciador de tarefas simples desenvolvido utilizando o framework Flask. Ele permite aos usuários criar, visualizar, atualizar e excluir tarefas de forma fácil e intuitiva.

## Funcionalidades
- Criar novas tarefas
- Visualizar a lista de tarefas
- Atualizar o status das tarefas
- Deletar tarefas
